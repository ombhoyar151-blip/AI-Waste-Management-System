# ♻️ Waste Classification Dashboard (EcoSort)

A smart and interactive *Waste Classification System* that helps users identify the type of waste and suggests the correct disposal method. This project promotes eco-friendly practices and efficient recycling.

---

## 🚀 Features

- 🧠 Smart waste classification (Organic, Plastic, Paper, Glass, Metal, Hazardous)
- 📊 Real-time dashboard with statistics:
  - Items Classified
  - Recycled Count
  - CO₂ Saved
  - Hazardous Waste Count
- 🔍 Input-based classification system
- 📋 Live table to track recent classifications
- 🎨 Modern and responsive UI design
- ⌨️ Enter key support for faster input

---

## 🛠️ Technologies Used

- *HTML5* – Structure of the application  
- *CSS3* – Styling and dashboard UI  
- *JavaScript* – Logic and dynamic updates
